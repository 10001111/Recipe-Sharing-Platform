# Recipe Sharing Platform - Frontend

Next.js frontend for the Recipe Sharing Platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The frontend will run on http://localhost:3000

## Configuration

- API URL: Configured in `lib/api.ts`
- Backend runs on: http://127.0.0.1:8000
- Frontend runs on: http://127.0.0.1:3000

## Development

- Pages are in `app/` directory (Next.js 13+ App Router)
- API client is in `lib/api.ts`
- Styles are in `app/globals.css`
